# 28-0 Standalone

A clean, Base44-free React/Vite version of the 28-0 Ryder Cup legends draft simulator.

## Run locally

1. Install Node.js from https://nodejs.org
2. Unzip this folder.
3. Open Terminal/Command Prompt in the folder.
4. Run:

```bash
npm install
npm run dev
```

5. Open the local URL shown, usually http://localhost:5173

## Build for web

```bash
npm run build
```

The finished website is created in the `dist` folder.

## Cheapest sharing options

- Vercel free Hobby: good for personal/non-commercial testing.
- Netlify free: also good for static React sites.
- GitHub Pages: free but slightly more fiddly for Vite.

For simple testing with mates, Vercel or Netlify is easiest.
